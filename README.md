## FAQ

**1) Does a No-Logs policy guarantee privacy?**  
No. It’s a start. The review looks for clear language, scope, and whether exceptions are spelled out.

**2) What does Kill Switch actually protect against?**  
Traffic leaks during disconnects. If the tunnel drops, the Kill Switch should block network traffic until the VPN is back.

**3) Will a VPN always improve speed?**  
No. It can reduce speed. What matters is consistency: server quality, routing, and how often you need to reconnect.

**4) Which server should I pick first?**  
Closest geographic server for baseline stability. Then test a second region if you need location access.
